#+title: Bang.el

DuckDuckGo-style bangs for emacs

* Installation
Either manually clone and load, or use [[https://github.com/radian-software/straight.el][straight.el]]:
#+begin_src emacs-lisp
  (use-package bang
    :straight
    (bang :host github :repo "Arian-D/bang.el"))
#+end_src
